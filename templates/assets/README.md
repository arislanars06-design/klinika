# Template images

The reception template (`../reception_template.docx`) embeds these two
images at build time:

| File | What it is | Suggested size |
|------|-----------|----------------|
| `logo.png` | Clinic logo (top-left of the letterhead) | 300–600 px square, PNG |
| `qr.png`   | QR code (top-right of the letterhead)   | 300–600 px square, PNG |

## Using your own real logo and QR

1. Replace either PNG with your file (keep the same file name).
2. Regenerate the template:

    ```
    python scripts/build_reception_template.py
    ```

3. Upload the new `templates/reception_template.docx` via the
   **Settings → Word shabloni** page, or just commit it to the repo.

## Regenerating the placeholders

The check-in defaults are auto-generated by
[`scripts/build_template_assets.py`](../../scripts/build_template_assets.py):

- `logo.png` — a stylised round "NurMed" clinic mark
- `qr.png`   — a real QR code linking to `https://instagram.com/nurmed_lor`

Re-run that script if you want to change the fallback branding.
